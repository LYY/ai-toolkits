# AI Toolkits

个人 AI agent skills 集合，跨平台共享，通过 [skills CLI](https://github.com/vercel-labs/skills) 分发。

## 安装

```bash
npx skills add LYY/ai-toolkits -g -y
```

## Skills

| Skill | 描述 |
|-------|------|
| [address-pr-comments](./skills/address-pr-comments/) | 处理 GitHub PR review comments，支持 human + AI bot 混合 review 场景 |
